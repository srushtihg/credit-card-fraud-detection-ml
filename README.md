# Credit Card Fraud Detection using ML + SHAP

## Overview
This project detects fraudulent credit card transactions using machine learning techniques.

## Features
- Handles class imbalance using SMOTE
- Uses Random Forest for classification
- Uses SHAP for explainability

## Dataset
Kaggle Credit Card Fraud Dataset

## Results
- Accuracy: ~95%
- ROC-AUC: ~0.95

## Tech Stack
- Python
- Scikit-learn
- SHAP
- Pandas

## How to Run
1. Open notebook in Google Colab
2. Install dependencies
3. Run all cells
